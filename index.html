📖 活动规则说明

一、参与流程

  1. 用户注册

    新玩家需要先注册，填写游戏ID并选择自己会玩的本体职业（心法）。可以选择多个职业。

  2. 活动报名

    已注册的玩家可以在"活动报名"页面点击报名按钮参与本次活动。

  3. 等待开组

    当报名人数达到6人或以上时，即可开始抽签分队。

二、组队配置

👥

标准配置

每个队伍固定3人配置：

  • 🗡️ 2名DPS（输出职业）
  • 💚 1名奶妈（治疗职业）

三、抽签机制

🎲 智能职业分配

系统会收集所有报名玩家的本体职业，组成职业池。抽签时：

  • 优先分配非本体职业，让大家体验不同的玩法
  • 只会分配职业池中的职业
  • 随机分组，确保公平性

四、特殊情况

⚠️ 注意事项

  • 人数不足：报名人数不足6人时无法开始抽签
  • 轮空机制：当人数不是3的倍数时，多余的玩家会轮空，等待下次活动
  • 本体冲突：如果玩家拥有职业池中的所有职业，可能会被分配到本体职业（会特别标注）

五、职业分类

🗡️ DPS职业

和尚、冰心、花间游、气纯、剑纯、天策、藏剑、毒经、鲸鱼、田螺、明教、丐帮、苍云、莫问、霸刀、蓬莱、凌雪、衍天、无方、刀宗、万灵、段氏

💚 奶妈职业

奶秀、奶毒、奶花、奶歌、奶药

💡 最后:Peace&Love，希望大家快乐竞技呦

⚠️ Firebase配置错误

请在代码中配置你的Firebase项目信息。步骤如下：

  1. 访问 Firebase Console
  2. 创建或选择一个项目
  3. 在项目设置中获取配置信息
  4. 将配置信息填入下面的代码中
const firebaseConfig = {
    apiKey: "你的API密钥",
    authDomain: "你的项目.firebaseapp.com",
    databaseURL: "你的数据库URL",
    projectId: "你的项目ID",
    storageBucket: "你的存储桶",
    messagingSenderId: "发送者ID",
    appId: "应用ID"
};
